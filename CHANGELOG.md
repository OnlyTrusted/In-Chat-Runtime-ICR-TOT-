# Changelog

All notable changes to this repository will be documented in this file.

## v0.1.0-spec — 2026-03-24

Initial public specification release.

Included in this release:

- `SYS1` boot contract and canonical state header
- `RTR` router with registry, transitions, watchdog, and chain handling
- `PROTO-EX` execution protocol
- `PROTO-AU-EX` audit protocol
- `INDEX` library with line references and patchnote index
- markdown documentation structure for public release
